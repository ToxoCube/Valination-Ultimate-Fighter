Online multiplayer

Static hosting (GitHub Pages, Netlify, itch.io embedded web, etc.) serves your game over HTTPS. Browsers then block plain ws:// as mixed content — you need a public wss:// WebSocket server (TLS). There is no “allow connect” permission to toggle; use the URL field below (or window.NETPLAY_SERVER) with wss://your-server…. For local testing only, open the HTML over http:// so ws://localhost:8080 can work.

Leave blank to use window.NETPLAY_SERVER (if set) or the game default. Changing this applies the next time you open Online or start a match.
